# Unemployement_ClassificationProject Overview
       This project focuses on analyzing the unemployment rate in India during the COVID-19 period. The dataset contains region-wise unemployment rates recorded over time. The goal of this project is to clean the data, visualize trends, and understand how unemployment changed during the pandemic.

Technologies Used
   Python
   Pandas
   NumPy
   Matplotlib
   Seaborn
   Google Colab
   Dataset

Unemployment_Rate_trailing_30_days.txt
   The dataset includes columns such as region, date, frequency, and unemployment rate.

Project Features
     Data cleaning and preprocessing
     Trend analysis
     Line charts and bar graphs
     Rural vs Urban unemployment comparison
     COVID-19 impact study
